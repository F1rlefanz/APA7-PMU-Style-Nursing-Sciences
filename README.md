APA 7th Edition PMU Style for Nursing Sciences (Zotero)

Dies ist eine angepasste Version des offiziellen APA 7-Zitationsstils, optimiert für die spezifischen Anforderungen des Instituts für Pflegewissenschaften der Paracelsus Medizinischen Privatuniversität (PMU).



📥 Download \& Installation

Klicke HIER, um die Datei apa-pmu.csl herunterzuladen.



Hinweis: Falls sich nur eine Textwüste im Browser öffnet: Rechtsklick auf den Link -> "Ziel speichern unter..." wählen.



🚀 Installation (3 Wege)

Weg A: Manuelle Installation (Empfohlen \& am sichersten)

Dieser Weg verhindert, dass der Stil bei Zotero-Updates plötzlich wieder verschwindet.



Öffne Zotero auf deinem Computer.



Gehe zu Einstellungen (Windows: Bearbeiten > Einstellungen | Mac: Zotero > Einstellungen).



Klicke auf den Reiter Erweitert und dort auf Dateien und Ordner.



Klicke auf den Button Datenverzeichnis anzeigen.



Öffne den Unterordner styles.



Kopiere die heruntergeladene Datei apa-pmu.csl direkt in diesen Ordner.



Starte Zotero ggf. neu.



Weg B: Der einfache Weg (Doppelklick)

Stelle sicher, dass Zotero geöffnet ist.



Mache einen Doppelklick auf die Datei apa-pmu.csl.



Bestätige die Abfrage "Willst du den Style installieren?" mit Ja.



Weg C: Über die Zotero-Oberfläche

Gehe in Zotero zu Einstellungen > Zitieren > Stile.



Klicke auf das Plus-Symbol (+) unter der Liste der Stile.



Wähle die Datei apa-pmu.csl aus und klicke auf Öffnen.



🛠 Layout-Feinschliff (Word 1,5-Zeilenabstand)

Um technische Fehler im Code zu vermeiden, wird der Zeilenabstand direkt in Microsoft Word gesteuert:



Formatvorlage finden: Klicke mit der rechten Maustaste in dein Literaturverzeichnis in Word.



Wähle Formatvorlagen > Formatvorlage ändern... (Suche nach „Bibliography“ oder „Zotero Bibliography“).



Abstand einstellen: Klicke auf Format > Absatz... und stelle den Zeilenabstand auf 1,5 Zeilen ein.



Zusatz-Tipp: Setze den „Abstand nach“ auf 12 pt, um die Quellen optisch sauber zu trennen.



Dauerhaft speichern: Wähle die Option „Neue auf dieser Vorlage basierende Dokumente“ und bestätige mit OK.
